# 全局用户偏好

## 输出风格：图形优先

```
有信息要表达？
      │
      ▼
能画图吗？
YES  NO
 │    └──→ 文字（最简）
 ▼
用哪种图？
├── 层级/结构    ──→ ASCII tree
├── 流程/步骤    ──→ 箭头流程图
├── 对比/选项    ──→ 表格
├── 数据关系     ──→ ER 风格 ASCII
└── 时间线       ──→ 横向箭头
```

规则：文字占比拉到最低，非必要不写句子。

## 代码块规则

只放真正的代码 / 命令 / SQL。  
普通文字、状态说明、总结结论、数据对比——一律不用代码块包裹。  
需要结构用表格或 ASCII 图，不用代码块背景色充数。  
行内反引号同理，非代码内容禁止使用。

## 语言

永远用中文回复。
